My Next Generation — новая перспектива на администрацию Minecraft серверов

MyNG (My Next Generation — "Моя Последующая Генерация") — это новый Minecraft Java проект, ориентированный на качественный игровой опыт, улучшение комьюнити, нестандартную генерацию и адекватную модерацию. Мы стремимся создать гармоничный и целостный мир, где каждый игрок найдёт своё место

Основные характеристики сервера

  • Нестандартная коммуникация

    Мы формируем среду, основанную на взаимном уважении и сотрудничестве. На сервере не допускается токсичное поведение, оскорбления и дискриминация.

  • Кастомная генерация мира

    Отход от стандартной карты Minecraft. На MyNG реализованы специальные биомы, ландшафты и сбалансированное распределение ресурсов для комфортного геймплея.

  • Профессиональная модерация

    Команда администрации обеспечивает соблюдение установленных правил и оказывает помощь игрокам. Все действия модераторов прозрачны и открыты для обсуждения.

  • Улучшенный игровой процесс

    Внедрение дополнительных механик, которые разнообразят игровой опыт, сохраняя при этом базовую атмосферу Minecraft. Отсутствие элементов pay-to-win.

Подключение к серверу

Параметры подключения

  • Сетевой адрес:
    play.myng.ru
  • Рекомендуемая версия клиента: 1.20.2
  • Поддержка более старых версий ограничена и может содержать меньше функциональных возможностей.
  • Обязательная регистрация: Да, через систему WhiteList

Процедура первого входа

  • 1. Заполните анкету для участия в WhiteList через форму на этом сайте.
  • 2. После получения подтверждения добавьте сервер в список мультиплеера.
  • 3. При первом подключении завершите процедуру регистрации непосредственно в игре.
  • 4. Ознакомьтесь с правилами сервера, доступными по ссылке на этом сайте.
  • 5. Для оперативной коммуникации рекомендуется присоединиться к Telegram-сообществу.

Организация коммуникации

Для координации игроков, публикации новостей и оказания поддержки используется платформа Telegram.

Основное сообщество:

Telegram канал: @MyNGcraft Telegram чат: @MyNGcraftChat

Структура коммуникации:

  • Telegram-канал (@MyNGcraft): Официальные объявления, информация об обновлениях сервера и предстоящих событиях.
  • Telegram-чат (@MyNGcraftChat): Общее обсуждение игрового процесса, вопросы и ответы, поиск партнёров для совместной игры.
  • Техническая поддержка и жалобы: Для сообщения о нарушениях правил, технических неполадках и спорных ситуациях используйте почту support@myng.ru
  • Вопросы по WhiteList: Обсуждение процесса подачи заявок и связанных с регистрацией вопросов.
Примечание: Сервер MyNG не использует платформу Discord для основной коммуникации. Все официальные информационные каналы, служба поддержки и общение сообщества сосредоточены в Telegram.

Система кланов

Кланы представляют собой организационные единицы с развитой инфраструктурой и собственными правилами.

Веб-сайты кланов

Каждому официально зарегистрированному клану предоставляется отдельный сайт, размещённый на GitHub Pages. Сайт доступен по субдомену проекта (например: clan.myng.ru/название-клана).

На сайте может публиковаться информация о клане, внутренние правила, список участников и новости.

Организационная структура

  • Лидер клана — несёт полную ответственность за деятельность организации.
  • Заместитель лидера — исполняет обязанности лидера в его отсутствие.
  • Офицеры — специализированные роли (координатор строительства, ответственный за рекрутинг и др.).
  • Рядовые участники — члены клана.

Лидер клана имеет право вводить новые внутриклановые должности по согласованию с администрацией сервера.

Правовые аспекты

Клан обязан разработать внутренний регламент, который не должен противоречить общим правилам сервера. Данный документ утверждается администрацияей и приобретает статус официального.

Процедура создания клана: Инициативная группа игроков обсуждает данный вопрос с представителями администрации в Telegram-сообществе.
